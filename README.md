## About

Learn Deutsch With Anes is a modern, interactive German learning website designed for absolute beginners (A1-A2 level). It features carefully graded short stories that help you learn German naturally through reading, vocabulary building, and comprehension quizzes.

### Features

- ЁЯУЪ **Graded Stories** - Carefully written A1-A2 level German texts
- ЁЯУЭ **Vocabulary Lists** - Learn key words with English translations
- тЭУ **Interactive Quizzes** - Test your comprehension after each story
- ЁЯОи **Neo-Bauhaus Design** - Clean, minimalist interface with high-contrast readability
- ЁЯУ▒ **Responsive** - Works beautifully on desktop and mobile devices
- тЪб **No Dependencies** - Pure HTML, CSS, and JavaScript

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required!

### Run Locally

1. Clone the repository:
```bash
git clone https://github.com/anesdevv/Learning-German-With-Anes.git
```

2. Open `index.html` in your browser:
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

### Deploy to Netlify

1. Push your code to GitHub
2. Go to [Netlify](https://app.netlify.com)
3. Click "Add new site" тЖТ "Import an existing project"
4. Select your GitHub repository
5. Netlify will auto-detect the HTML site and deploy it!
