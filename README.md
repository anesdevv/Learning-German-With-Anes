## About

Learn Deutsch With Anes is a modern, interactive German learning website designed for absolute beginners (A1-A2 level). It features carefully graded short stories that help you learn German naturally through reading, vocabulary building, and comprehension quizzes.

### Features

- 📚 **Graded Stories** - Carefully written A1-A2 level German texts
- 📝 **Vocabulary Lists** - Learn key words with English translations
- ❓ **Interactive Quizzes** - Test your comprehension after each story
- 🎨 **Neo-Bauhaus Design** - Clean, minimalist interface with high-contrast readability
- 📱 **Responsive** - Works beautifully on desktop and mobile devices
- ⚡ **No Dependencies** - Pure HTML, CSS, and JavaScript

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required!

### Run Locally

1. Clone the repository:
```bash
git clone https://github.com/anesdevv/Learning-German-With-Anes.git
```

2. Open `index.html` in your browser:
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```
